# 1431.拥有最多糖果的孩子
```c++ {.line-numbers}
class Solution {
public:
    vector<bool> kidsWithCandies(vector<int>& candies, int extraCandies) {
        int max = 0;
        for(int i : candies){
            if(i > max)
                max = i;
        }
        vector<bool> ans;
        for(int i = 0; i < candies.size(); i++){
            ans.push_back(candies[i] + extraCandies >= max);
        }
        return ans;
    }
};
```
**C++ max_element()的使用**   
max_element用来查询最大值所在的第一个位置   
max_element获取的是地址   
*max_element获取的是位置，从1开始计数
