# next数组不同的实现方法
文本串 aabaabaaf   
模式串 aabaaf   
前缀表   
a a b a a f   
两种方式：   
0 1 0 1 2 0   
-1 0 1 0 1 2   
# 具体代码
## 求next数组
```c++ {.line-numbers}
void getNext(string s, int next[]) {
    // 1. 初始化
    // i指向后缀末尾位置，j指向前缀末尾位置
    j = 0;
    next[0] = 0;
    
    for(i = 1; i < s.size(); i++){
        // 2. 处理前后缀不相同的情况
        while(j > 0 && s[i] != s[j])
            // j向前回退
            j = next[j-1];

        // 3. 处理前后缀相同的情况
        if(s[i] == s[j])
            j++;
        
        // 4. 更新next数组
        next[i] = j;
    }

    
```
