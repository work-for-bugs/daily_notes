# 前缀与后缀
什么是前缀？ 包含首字母不包含尾字母的所有子串   
a aa aab aaba aabaa   
什么是后缀？ 包含尾字母不包含首字母的所有子串   
f af aaf baaf abaaf   
# 最长相等前后缀
求最长相等前后缀   
a       0   
aa      1   
aab     0   
aaba    1   
aabaa   2   
aabaaf  0

# 使用前缀表的匹配过程
到某个字符匹配失败，就看它前面的子串的前缀表，如果它前面有相同的前后缀，就从它的前缀表中找到相同的前后缀，然后从相同前缀后面开始匹配
