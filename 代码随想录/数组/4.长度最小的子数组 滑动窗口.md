# 滑动窗口
for循环的索引下标是滑动窗口里的**终止位置**   
起始位置需要用移动策略来操作
# 如何移动起始位置
**什么时候移动起始位置**   
区间内元素和大于等于target时移动起始位置   
```c++ {.line-numbers}
i = 0;
for(j=0;j<nums.size;j++){
    sum += nums[j];
    while(sum >= s){//持续向后移动的过程，直到窗口内元素和小于s
        subL = j - i + 1;
        result = min(result, subL);
        //移动起始位置
        sum = sum - nums[i];
        i++;
    }
    return result;
}
```

与暴力解法的区别在于，暴力的双层for循环中，第一层循环的索引下标是滑动窗口的**起始位置**。  
而滑动窗口的循环索引下标是**终止位置**
