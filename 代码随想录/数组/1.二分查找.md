区间是左闭右闭   [ ]
还是左闭右开     [ )

左闭右闭
```c++ {.line-numbers}
left = 0
**right = nums.size - 1**
while(left **<=** right)  //区间是否合法  left == right在区间里合法
{
    middle = left + (right - left) / 2;
    if(nums[middle] > target)
        **right = middle - 1;**
    else if(nums[middle] < target)
        left = middle + 1;
    else
        return middle;
}
return -1;
```

左闭右开
```c++ {.line-numbers}
left = 0
**right = nums.size**
while(left **<** right)  //区间是否合法  left == right在区间里不合法
{
    middle = left + (right - left) / 2;
    if(nums[middle] > target)
        **right = middle;**
    else if(nums[middle] < target)
        left = middle + 1;
    else
        return middle;
}
return -1;
```
