# 整体思路
遇到左括号就入栈右括号，遇到右括号的时候直接弹出作比较。   
```c++ {.line-numbers}
stack<char> st;
if(s.size() % 2 != 0)
    return false;
for(int i = 0; i < s.size(); i++){
    if(s[i] == '('){
        st.push(')');
    else if(s[i] == '{')
        st.push('}');
    else if(s[i] == '[')
        st.push(']');
    else if(st.empty() || st.top() != s[i])
        return false;
    else
        st.pop();
}
    return st.empty();
```
