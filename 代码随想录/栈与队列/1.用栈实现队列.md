```c++ {.line-numbers}
void push(int x){
    stackIn.push(x);
}
int pop(){
    if(stackOut.empty()){
        while(!stackIn.empty()){
            stackOut.push(stackIn.top());
            stackIn.pop();
        }
        result = stackOut.top();
        stackOut.pop();
        return result;
    }
}  
```
队列                3 4 5   
stackIn            5   
stackOut          4 3   
pop 1 2   
pop的时候，stackOut里非空，就从stackOut里pop，如果stackOut空了，就把stackIn里的元素都放到stackOut里。也就是说，只有在stackOut为空时，才把stackIn里的元素都放到stackOut里。   
stack声明：stack<int> stackIn, stackOut;
