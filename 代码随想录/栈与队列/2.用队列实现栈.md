# 整体思路
把size-1个元素弹出来，再压入队列   
```c++ {.line-numbers}
void push(int x){
    que.push(x);
}

int pop(){
    size = que.size();
    size--;
    while(size--){
        que.push(que.front());
        que.pop();
    }
    int res = que.front();
    que.pop();
    return res;
}

int top(){
    return que.back();
}

bool empty(){
    return que.empty();
}
```
