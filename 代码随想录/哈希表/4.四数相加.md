# 整体思路
4个数组，不考虑去重   
a+b放到一个集合，c+d放到一个集合   

# map解题过程
a+b放到key，出现的次数放到value，遍历c+d，判断0-（c+d）是否出现   
把n^4拆成两个n^2的时间复杂度   

# 代码实现
```c++ {.line-numbers}
unordered_map<int, int> map;

for (a : A) {
    for (b : B) {
        map[a + b]++;
    }
}

for (c : C) {
    for (d : D) {
        if (map.find(0 - (c + d)) != map.end()) {
            res += map[0 - (c + d)];
        }
    }
}

return res;
```
