高度和深度   
高度 根节点最大  深度  叶子节点最大   
求高度  后序  左右中   
求深度  前序  中左右   
```c++ {.line-numbers}
int getHeight(TreeNode* node){
    if(node == nullptr)
        return 0;
    int leftHeight = getHeight(node->left);
    int rightHeight = getHeight(node->right);
    int height = 1 + max(leftHeight, rightHeight);
    return height;
}
```
