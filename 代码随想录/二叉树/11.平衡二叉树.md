# 高度、深度、遍历顺序
求高度用后序遍历   
求深度用前序遍历   
# 代码实现
```c++ {.line-numbers}
int getHeight(TreeNode* root) {
    // 终止条件
    if (!root) return 0;
    int leftHeight = getHeight(root->left);
    if(leftHeight == -1)  // 左子树不平衡
        return -1;
    int rightHeight = getHeight(root->right);  // 右子树不平衡
    if(rightHeight == -1)
        return -1;
    return abs(leftHeight - rightHeight) > 1 ? -1 : 1 + max(leftHeight, rightHeight);
}
```
