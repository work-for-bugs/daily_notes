# 两两交换链表中的节点
```c++ {.line-numbers}
dummyHead->next = head;
cur = dummyHead;
while (cur->next != nullptr && cur->next->next != nullptr) {
    temp = cur->next;
    temp1 = cur->next->next->next;
    cur->next = cur->next->next;
    cur->next->next = temp;
    temp->next = temp1;
    cur = cur->next->next;
}
return dummyHead->next;
```
