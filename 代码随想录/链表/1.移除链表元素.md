# 移除链表中的元素
两种方法   
## 原链表删除元素
```c++ {.line-numbers}
// 删除头结点
while(head != null && head->val == val)
    head = head->next;
// 删除非头结点
cur = head;
while(cur != null && cur->next != null){
    if(cur->next->val == val)
        cur->next = cur->next->next;
    else
        cur = cur->next;
}
return head;
```
## 使用虚拟头结点
dummy head 虚拟头结点   
所有结点可以用同样的规则来删除   
```c++ {.line-numbers}
ListNode* dummyHead = new ListNode(0);
dummyHead->next = head;
ListNode* cur = dummyHead;
while(cur->next != null){
    if(cur->next->val == val){
        ListNode* tmp = cur->next;
        cur->next = cur->next->next;
        delete tmp;//C++要释放空间
    }
    else
        cur = cur->next;
}
return dummyHead->next;
```
